# 🎙 Real-Time Multilingual Voice AI Agent

## 🚀 Overview
This project is a real-time voice-based AI system for booking clinical appointments.
It supports multilingual conversations and maintains contextual memory.

## ⚡ Features
- Voice input & output
- Appointment booking / rescheduling / cancellation
- Multilingual (English, Hindi, Tamil)
- Context-aware conversations
- Low-latency streaming pipeline

## 🧠 Architecture
Mic → STT → LLM → Scheduler → TTS → Audio Output

## ⚙️ Tech Stack
- FastAPI (Backend)
- WebSockets (Realtime)
- Whisper (STT)
- OpenAI (LLM)
- ElevenLabs (TTS)
